# 🛍️ Gestión de Productos (Node.js Nativo)

Este proyecto es un servidor web minimalista que gestiona un inventario mediante archivos de texto, utilizando operaciones no bloqueantes.

## ⚙️ Características
- **Servidor:** HTTP nativo (sin frameworks externos).
- **Persistencia:** File System (`fs.promises`) con lectura y escritura asíncrona.
- **Frontend:** Manipulación dinámica del DOM y lógica de ordenamiento en JS vanilla.

## 🚀 Cómo ejecutar
1. Iniciar servidor: `node server.js`
2. Abrir `index.html` en el navegador.
